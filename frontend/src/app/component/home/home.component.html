Technical Assessment Overview

Objective: Your goal is to create a Spring Boot application with Java to populate an SQL database with provided insurance data and then develop an Angular frontend application to fetch and display this data.

Instructions

Backend (Spring Boot with Java):

  • Set up a relational SQL database of your choice.
  • Create a structure that can accommodate the provided JSON data format.
  • Implement a function in your Spring Boot application to fill in the database with the provided data upon startup.
  • Make sure your application can connect to the database only using information from the application.properties file.
  • Design RESTful APIs to fetch the data from the database for CRUD operations in the frontend.

Frontend (Angular):

  • Create an Angular application with two pages (Design is up to you - Show us your skills):
    • Table Page: Display the insurance data in a table format with sorting capabilities on key information of your choosing.
    • Card Page: Showcase the insurance data using cards (Think about initial response times of your requests!!).
  • Ensure data is fetched from your backend using the APIs you've developed.
  • While responsiveness isn't mandatory, we appreciate attention to detail and good design practices.

GitHub

Push your Spring Boot application and Angular application to separate GitHub repositories.

Ensure that your README.md files in both repositories provide clear instructions on how to set up and run the projects.

Technology Stack

Backend: Java, Spring Boot, SQL database (of your choice)

Frontend: Angular

Version Control: Git, GitHub

Deliverables

  • Link to the GitHub repository for the Spring Boot application.
  • Link to the GitHub repository for the Angular application.
  • A brief report or README detailing any challenges you faced and the decisions you made during the implementation.
